Android App Guide

This is a basic guide to the best Android Apps to store your Boarding Passes.

-----------------------------------------
--- Airline's Own App ---

Pros:
+ Most accurate for live time information on Arrivals, Departures, Terminals, Gates, and Delays
+ Most apps can store your Boarding pass

Cons:
- You will need to download an app for each airline you use
- Cannot be used to test demonstration boarding passes

-----------------------------------------
--- KDE Itinerary ---

Pros:
+ Open Source
+ Easy to share and transfer data between devices via .itinerary files
+ Can also be used to plan trips

Cons:
- Can be slightly difficult to use for first time users
- There is no live time flight information (Terminals, Gates, Delays Etc ...)

-----------------------------------------
--- Flight Boarding Pass Wallet ---

Pros:
+ Attractive design
+ Can show boarding pass in AZTEC, PDF417, and QR
+ Can show details of more than one leg of a flight

Cons:
- Proprietary, and the free version has ads
- Reports of some flights being displayed departing one day earlier

-----------------------------------------
--- MyFlights ---

Pros:
+ Quick and easy to use

Cons:
- You will need an account to use the app

-----------------------------------------
--- Flight Status + Boarding Pass ---

Pros:
+ Quick and easy to use

Cons:
- Cannot import barcode from images (.png .jpg Etc ...)
- If the flight does not exist in the app's database the pass gets rejected
- Inaccurate live time flight information for multi-leg flights on one flight number

-----------------------------------------
--- Fobi PassWallet ---

Pros:
+ Quick and easy to use
+ Can be used for other purposes

Cons:
- The boarding pass has a yellow design background and this cannot be changed
- There is no live time flight information (Terminals, Gates, Delays Etc ...)

-----------------------------------------
--- PassAndroid ---

Pros:
+ Open Source
+ Data can be exported and shared with other PassAndroid users using open source .espass files
+ Can be used for other purposes

Cons:
- Designs are very basic
- There is no live time flight information (Terminals, Gates, Delays Etc ...)
- Details about the flight must be entered manually

-----------------------------------------
--- Technofy Infotech Flight Boarding Pass ---

Pros:
+ Can show boarding pass in AZTEC, PDF417, and QR
+ Has a World Clock Converter

Cons:
- Proprietary, and the app has ads
- There is no live time flight information (Terminals, Gates, Delays Etc ...)

-----------------------------------------
--- Pass2U ---

Pros:
+ Can be used for other purposes

Cons:
- Proprietary, and the free version has ads
- There is no live time flight information (Terminals, Gates, Delays Etc ...)
- Details about the flight must be entered manually

-----------------------------------------
--- Catima ---

Pros:
+ Open Source and can be used for other purposes

Cons:
- There is no live time flight information (Terminals, Gates, Delays Etc ...)
- Flight details must be entered manually as basic notes

The apps listed here are general QR barcodes for any use. You can import boarding passes with no problems, but you will have to
enter your flight details manually as basic notes, and there is no live time flight information.

The best way to do this is to import the boarding pass barcodes, and use them as a backup. Then you can try to transfer the codes
to one of the other Boarding Pass apps listed above, and hope for the best.
